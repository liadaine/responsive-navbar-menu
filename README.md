# 📌 Menu de Navegação Responsivo

Projeto desenvolvido com HTML e CSS criando um menu de navegação responsivo, com adaptação para dispositivos móveis e interação sem uso de JavaScript.

---

## 🛠️ Tecnologias utilizadas
- HTML
- CSS

---

## 🎯 Objetivo
Praticar a criação de menus de navegação modernos, trabalhando responsividade, organização de layout e interações utilizando apenas CSS.

## 📚 O que foi praticado
- Estruturação de navegação em HTML  
- Estilização com CSS  
- Criação de navbar responsiva  
- Implementação de menu mobile (hamburguer)  
- Interação sem uso de JavaScript  
- Uso de media queries para responsividade
